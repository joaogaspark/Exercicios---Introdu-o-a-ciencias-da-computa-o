# Exercicios---Introduçao-ciencias-da-computação
Projeto desenvolvido em linguagem C para a disciplina de Introdução à Ciência da Computação da UNIVALI. O programa cadastra até 5 alunos, calcula a média de três notas, informa a situação de cada aluno (aprovado ou reprovado) e identifica a maior média da turma, utilizando vetores, funções, estruturas de repetição e condicionais.
